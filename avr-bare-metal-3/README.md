# AVR Bare Metal #3

This repository contains the source code used in the article:

**[Blinking a LED using Timer Interrupt Method | AVR Bare Metal #3](https://bitstosystems.hashnode.dev/avr-bare-metal-3)** 

## Files

* `main.c` - Application source code
* `Makefile` - Build and flashing commands

## Build

```bash
make
```

## Flash

```bash
make flash
```

## Other Files
- The `.elf` and `.hex` files are the output of the  `make` command
- `note_makefile.txt` - Contains my understanding of Makefile.
